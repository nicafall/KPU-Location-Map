# KPU Surrey Library Distance Map

## Description
This web application dynamically displays the user's current location on a map and calculates the distance to the KPU Surrey Library using the Haversine Formula.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Leaflet.js
- OpenStreetMap
- Browser Geolocation API

## Features
- Detects the user's current location.
- Displays the user's location and the KPU Surrey Library on a map.
- Calculates the distance between the two locations.
- Updates the map dynamically when the user's location changes.
